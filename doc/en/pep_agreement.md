## 1. SUBJECT OF THE AGREEMENT
1.1. This Agreement defines the procedure for electronic interaction between the Operator and the User within the Automated System (LMIS).
1.2. The Parties agree that electronic documents signed by the User in the System using a **Simple Electronic Signature (SES)** — a unique 6-digit PIN code entered on the User's authenticated personal device — are legally recognized as equivalent to paper documents signed with the User's handwritten signature.

## 2. RULES FOR USING SES
2.1. The Simple Electronic Signature (SES) key consists of the User's unique System ID and their secret PIN code.
2.2. The User independently creates their PIN code in the System and undertakes to keep it strictly confidential from third parties.
2.3. Entering the PIN code by the User in the System interface (including via a personal smartphone or self-service terminal) after reviewing the text of an electronic document (Order, Job Search Application, etc.) constitutes the User's full and unconditional agreement with the contents of said document.

## 3. RESPONSIBILITIES OF THE PARTIES
3.1. The User bears personal responsibility for the security of their SES key (PIN code). All actions performed in the System using the User's SES are deemed legally significant and to have been performed by the User themselves.
3.2. In the event of compromise (loss, disclosure) of the PIN code, the User is obliged to immediately notify the Operator by personally visiting the Employment Center to reset the authorization key.
3.3. The Operator undertakes to securely log the User's actions and ensure the absolute immutability of electronic documents after they are signed via SES.

## 4. VALIDITY PERIOD
4.1. This Agreement enters into force from the moment it is signed by both Parties on paper and remains valid indefinitely.
