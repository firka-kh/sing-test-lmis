# Verification Portal (LMIS Mockup)

This is the Identity Verification Portal project for the Ministry of Labor, Migration and Employment of the Republic of Tajikistan.
Это проект Портала верификации личности для Министерства труда, миграции и занятости населения РТ.

### Running Locally / Запуск локально
1. Make sure you have Java JDK 17+ installed.
2. Run the server using Maven Wrapper:
```bash
./mvnw spring-boot:run
```
3. Open `http://localhost:8080/` in your browser.
